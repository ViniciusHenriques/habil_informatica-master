function ConsultarDistribuicaoDFe(const aCodigoUF, aCNPJCPF : string; const aNSU: string = '000000000000000'; const aTipoNSU: TNSUKind = nkUltimo): string;

Função de "Consulta da Relação de Documentos a serem distribuidos para determinado CPF ou CNPJ de acordo com a NF-e." 

    Ex:
    Por CPF utilizando Ultimo NSU:
    _Result := ConsultarDistribuicaoDFe('41', '12345678900');

    Por CNPJ utilizando NSU especifico:
    _Result := ConsultarDistribuicaoDFe('41', '12345678912345', '000000000000015', nkEspecifico);

Parâmetros
aCodigoUF
- Código da UF que sera feita a consulta DF-e. Ex: 41 (PARANÁ)
aCNPJCPF
- CPF ou CNPJ que solicitou a Distribuição DF-e. Ex: "12345678912" ou "12345678911234"
aNSU
- Número NSU (Número Sequencial Unico) para consulta. Ex: "000000000000001"
aTipoNSU
- Tipo do NSU, o tipo que será utilizado para requisição Especifico ou o Ultimo. Ex: nkEspecifico
Retornos

Retorna Os XMLs Destinados pela busca do NSU.

Exceções
EspdNFeDFeUFInvalidaException
quando o código da UF configurada é inválido
EspdNFeDFeCNPJInvalidoException
quando CNPJ configurado é inválido
EspdNFeDFeCPFInvalidoException
quando CPF configurado é inválido
EspdNFeDFeNSUInvalidaException
quando NSU configurado é inválido
EspdNFeDFeException
quando houve algum problema na requisição.
